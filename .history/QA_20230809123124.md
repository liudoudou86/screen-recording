```css
.method-button {
  margin-top: 20px;
  height: 120px;
  width: 130px;
  display: flex;
  flex-direction: column-reverse;
  align-items: center;
  .icon-span {
    margin-top: 5px;
  }
}
```

将这段CSS代码插入到`<style>`标签中的`.main_app`样式之后即可。
